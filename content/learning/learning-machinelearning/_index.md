## 机器学习算法

| 监督学习(supervisor learning)           | 分类(classification)\回归(regression) |
| --------------------------------------- | ------------------------------------- |
| k-临近算法                              | 线性回归                              |
| 朴素贝叶斯算法                          | 局部加权线性回归                      |
| 支持向量机(support vector machine, SVM) | Ridge回归                             |
| 决策树                                  | Lasso最小回归系数估计                 |

| 无监督学习 (unsupervisor learning) | 聚类(clustering) |
| ---------------------------------- | ---------------- |
| k-均值                             | 最大期望算法     |
| DBSCAN                             | Parzen窗设计     |
